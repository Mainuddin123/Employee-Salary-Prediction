# 💼 Employee Salary Prediction

A Machine Learning web application built using **Python**, **Scikit-Learn**, and **Streamlit** to predict employee salaries based on demographic and professional information.

---

## 📌 Project Overview

This application predicts employee salaries using Machine Learning. It helps estimate salaries based on employee attributes such as age, experience, education level, department, job title, gender, and location.

---

## 🚀 Features

- Predict employee salary instantly
- Interactive Streamlit web application
- Decision Tree Regressor (Best Model)
- Hyperparameter tuning using GridSearchCV
- Professional dark-themed user interface
- Real-time salary prediction

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

---

## 🤖 Machine Learning Models

- Decision Tree Regressor ✅ (Final Model)
- KNN Regressor (Compared)

The Decision Tree Regressor achieved the best performance and was selected for deployment.

---

## 📊 Model Performance

| Metric | Value |
